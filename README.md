# tnx.light

## Local start instruction
* start the docker containers using

    ``docker compose -f docker.compose.dev.yml up --build ``

* frontend can be accessed via

    ``http://localhost:8080``

* backend swagger docs via

    ``http://localhost:4000/docs``
